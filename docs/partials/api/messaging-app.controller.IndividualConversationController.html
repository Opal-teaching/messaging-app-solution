IndividualConversationController
controller in module messaging-app

Description

Manages ./views/messages/individual-conversation.html

Dependencies

  • MessengerService
  • $timeout

Methods

  • deleteConversation()

    Makes a call to MessengerService to delete the current conversation in the page, if this fails, it displays an alert otherwise, it pops the current page.

  • initController()

    Initializes the controller

  • sendMessage()

    Calls the MessengerService to add a message to the conversation array, updates vm.conversations after.

Properties

  • conversation

    Contains the current conversation in the page.

  • messageContent

    Contains the new message contents

  • noMessages

    Flag to display "No messages" in case of an empty conversation
