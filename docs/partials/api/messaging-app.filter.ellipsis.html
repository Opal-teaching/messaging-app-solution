ellipsis
filter in module messaging-app

Description

If the text is larger than maxLength, shortened to maxLength and apply ellipsis

Usage

In HTML Template Binding

{{ ellipsis_expression | ellipsis:maxLength }}

In JavaScript

$filter('ellipsis')(text, maxLength)

Parameters

ParamTypeDetails
textstring

Text to be processed

maxLengthstringnumber

Maximum length of text

Returns

function

Returns function transformation
