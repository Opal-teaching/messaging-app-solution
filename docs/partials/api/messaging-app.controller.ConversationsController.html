ConversationsController
controller in module messaging-app

Description

Manages ./views/messages/conversations.html

Dependencies

  • MessengerService
  • $timeout
  • $scope

Methods

  • goToConversation()

    Pushes the new-conversation.html page onto the stack with the selected conversation as parameter

  • initController()

    Initializes the controller and the scope variables

  • initializeEvents()

    Initializes navigator event to refresh conversations

  • newConversation()

    Pushes the new-conversation.html page onto the stack using a 'lift' animation

Properties

  • conversations

    Contains array of conversations

  • emptyConversations

    Flag to determine when to show 'No message'

  • searchConversationString

    String used to search conversations, it matches names

Events

  • $destroy

    Listens to the controller destruction and kills event listeners.
